# kanban team member-remove

**Command:** `kanban team member-remove <team_id> <user_id>`

## Description

Remove member from team.

## Arguments & Options

- `team_id`: Team ID
- `user_id`: User ID to remove

## Examples

```bash
kanban team member-remove <team_id> <user_id>
```
