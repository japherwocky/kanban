# kanban team list

**Command:** `kanban team list [--org_id|-o <value>]`

## Description

List teams in an organization.

## Arguments & Options

- `--org_id` / `--o` (required): Organization ID (required)

## Examples

```bash
kanban team list [--org_id|-o <value>]
```
