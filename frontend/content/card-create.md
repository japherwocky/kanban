# kanban card create

**Command:** `kanban card create <column_id> <title> [--description|-d <value>] [--position|-p <value>]`

## Description

Create a new card.

## Arguments & Options

- `column_id`: Column ID
- `title`: Card title
- `--description` / `--d`: Card description
- `--position` / `--p` (default: `0`): Position

## Examples

```bash
kanban card create <column_id> <title> [--description|-d <value>] [--position|-p <value>]
```
