# kanban login

**Command:** `kanban login <u> [--password|-p <value>] [--server|-s <value>]`

## Description

Login to the Kanban server.

## Arguments & Options

- `u`: s
- `--password` / `--p` (required): Password
- `--server` / `--s` (default: `http://localhost:8000`): Server URL

## Examples

```bash
kanban login <u> [--password|-p <value>] [--server|-s <value>]
```
