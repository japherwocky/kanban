# kanban column create

**Command:** `kanban column create <board_id> <name> <position>`

## Description

Create a new column.

## Arguments & Options

- `board_id`: Board ID
- `name`: Column name
- `position`: Position

## Examples

```bash
kanban column create <board_id> <name> <position>
```
