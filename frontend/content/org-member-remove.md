# kanban org member-remove

**Command:** `kanban org member-remove <org_id> <user_id>`

## Description

Remove member from organization.

## Arguments & Options

- `org_id`: Organization ID
- `user_id`: User ID to remove

## Examples

```bash
kanban org member-remove <org_id> <user_id>
```
