# kanban card update

**Command:** `kanban card update <card_id> <title> [--description|-d <value>] [--position|-p <value>] [--column|-c <value>]`

## Description

Update a card.

## Arguments & Options

- `card_id`: Card ID
- `title`: Card title
- `--description` / `--d`: Card description
- `--position` / `--p`: Position
- `--column` / `--c`: New column ID

## Examples

```bash
kanban card update <card_id> <title> [--description|-d <value>] [--position|-p <value>] [--column|-c <value>]
```
