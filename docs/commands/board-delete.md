# kanban board-delete

Delete a board.

## Usage

```bash
kanban board-delete <board_id>
```

## Arguments

- `board_id`: Board ID to delete

## Warning

This action is **irreversible**. All columns and cards in the board will be permanently deleted.

## Examples

```bash
kanban board-delete 1
```
