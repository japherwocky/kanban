# kanban boards

List all accessible boards.

## Usage

```bash
kanban boards
```

## Description

Lists all boards you have access to, including:
- Boards you own
- Boards shared with your teams
- Boards public to your organizations

## Output

```
ID    Name                 Sharing
----  -------------------  --------------------------------
1     Project Alpha        (shared with team 2)
2     Sprint Planning      (public to organization)
3     Personal Tasks
```

## Examples

```bash
# List all boards
kanban boards
```
