## Authentication

- [`kanban config`](/docs/commands/config) - Configure the CLI or show current settings
- [`kanban login`](/docs/commands/login) - Login to the Kanban server
- [`kanban logout`](/docs/commands/logout) - Logout and clear credentials

## Boards

- [`kanban boards`](/docs/commands/boards) - List all boards
- [`kanban board-create`](/docs/commands/board-create) - Create a new board
- [`kanban board`](/docs/commands/board) - Show board details
- [`kanban board-delete`](/docs/commands/board-delete) - Delete a board
- [`kanban share`](/docs/commands/share) - Share board with team or make private
